# Skillset
This skillset focuses on estimating force field parameters and coarse-grained (CG) bead sizes/diameter (σ) for molecular simulations. It emphasizes geometry-based sigma estimation, bond effects, and how parameters influence system properties. Use this for accurate, efficient CG modeling—do not directly copy all-atom parameters to CG, as they must account for averaged degrees of freedom, entropy, and many-body effects.

## 1. Core Force Field Parameters
### 1.1 Bonded Parameters
- **Bonds**: `Type1-Type2 k_bond (kcal/mol/Å²) r₀ (Å)`
  - Energy: `E = k_bond × (r - r₀)²`
  - Examples (All-Atom): C-C 310.0 1.526; C=O 570.0 1.229
  - Effects: Higher k_bond → stiffer bonds, smaller vibrations, higher frequency modes. Larger r₀ → bigger molecule size, lower density.

- **Angles**: `Type1-Type2-Type3 k_angle (kcal/mol/rad²) θ₀ (deg)`
  - Energy: `E = k_angle × (θ - θ₀)²`
  - Examples (All-Atom): C-C-C 40.0 109.5; O=S=O 120.0 120.0
  - Effects: Higher k_angle → rigid shapes, less bending. θ₀ changes geometry (e.g., tetrahedral vs. planar), affecting packing and diffusion.

- **Dihedrals**: `Type1-Type2-Type3-Type4 k_dih (kcal/mol) n δ (deg)`
  - Energy: `E = k_dih × [1 + cos(nφ - δ)]`
  - Examples (All-Atom): C-C-C-C 0.15 3 0.0; C-C=C-C 1.50 2 180.0
  - Effects: Higher k_dih → higher rotation barriers, fewer conformers, slower dynamics. n/δ control symmetry and preferred states, influencing isomerization and flexibility.

- **Impropers**: `Type1-Type2-Type3-Type4 k_imp (kcal/mol/rad²) ψ₀ (deg)`
  - Energy: `E = k_imp × (ψ - ψ₀)²`
  - Examples (All-Atom): C-C-O-O 10.5 0.0
  - Effects: Maintains planarity; higher k_imp → flatter structures, better for aromatics/peptides, reducing out-of-plane fluctuations.

### 1.2 Nonbonded Parameters
- **Lennard-Jones (VDW)**: `Type ε (kcal/mol) Rmin/2 (Å) or σ (Å)`
  - Energy: `U = ε[(Rmin/r)¹² - 2(Rmin/r)⁶]` (where, Rmin ≈ 1.122 × σ)
  - Examples (All-Atom): C -0.110 2.000; O -0.120 1.700
  - Effects: Higher ε → stronger attractions, higher density, viscosity, heat of vaporization (ΔH_vap), lower diffusion. Larger σ/Rmin → bigger excluded volume, lower density, higher compressibility, shifted RDF peaks.

- **Charges**: `Atom q (e)`
  - Energy: `E = k_e × q₁q₂ / r`
  - Examples (All-Atom): O_water -0.834; S_sulfone 1.12
  - Effects: Larger |q| → stronger electrostatics, better solvation/H-bonds, higher melting points, altered phase behavior.

**Parameter Dependencies Summary**:
| Parameter | Controls | Structure Effect | Dynamics Effect | Property Changes if Increased |
|-----------|----------|------------------|-----------------|-------------------------------|
| r₀ | Bond length | Larger size | Vibration center | ↓ Density, ↑ Diffusion |
| k_bond | Bond stiffness | Less stretch | High-freq modes | ↓ Vibration amplitude, ↑ Frequency |
| θ₀ | Angle shape | Geometry change | Bending motions | Alters packing (e.g., bent → compact) |
| k_angle | Angle stiffness | Less bend | Medium-freq modes | ↓ Flexibility, ↑ Rigidity |
| k_dih | Rotation barrier | Conformers | Isomerization | ↓ Conformational space, ↓ Dynamics |
| ε | Attraction strength | Packing | Cohesion | ↑ Density, ↑ Viscosity, ↑ ΔH_vap, ↓ Diffusion |
| σ/Rmin | Size/exclusion | Spacing | Collisions | ↓ Density, ↑ Compressibility, ↑ Pressure |
| q | Polarity | Interactions | H-bonds | ↑ Solvation energy, ↑ Melting point |

## 2. CG Bead Sigma (σ) Estimation from Bond Geometry
**Core Idea**: σ_bead represents effective bead size of the grouped atoms, based on atom spread (from bonds/angles) + corrections. Longer/flexible bonds → larger σ. 
Use for CG models; always re-optimize vs. all-atom.

**Inputs (All-Atom)**:
- Atoms: element, mass (amu), σ_atom (Å) 
- Bonds: lengths (Å), orders (1=single, 2=double, 3=triple)
- Angles (deg)

**Algorithm**:
1. Build 3D positions (place atom 1 at [0,0,0]; add others via bonds/angles).
2. Compute COM: `r_COM = Σ(m_i * r_i) / Σm_i`
3. Compute R_g: `R_g² = Σ(m_i * |r_i - r_COM|²) / Σm_i`; `R_g = sqrt(R_g²)`
4. Compute σ_bead: `σ_bead = 2 * R_g + w_VDW * σ_avg + f_flex + f_elec`
   - σ_avg = average σ_atom
   - w_VDW = 0.3-0.6 (higher for compact groups)
   - f_flex = flexibility (0.25 single, 0.1 double, 0.05 triple)
   - f_elec = electronic (0.02-0.05 * base for multiple bonds)

**Bond Order Effects**:
| Bond Type | Length | Flexibility | σ Correction | Notes |
|-----------|--------|-------------|--------------|-------|
| Single | Long (1.5Å) | High | +0.2-0.3Å | Spread out, flexible → larger σ |
| Double | Medium (1.3Å) | Medium | +0.1Å +3% | Rigid, electron density → medium σ |
| Triple | Short (1.2Å) | Low | +0.0Å +5% | Compact, rigid → smaller σ |

Effects: Higher bond order → shorter/stiffer → smaller R_g → smaller σ (Δσ ≈0.2-0.3Å per order). Changes σ by 10-15%, impacting density/packing.

## 3. Worked Examples
### Nitrite Ion (NO₂⁻): Bent, 2 bonds (order 1.5), angle 115°
- All-Atom Positions: N[0,0,0], O1[1.24,0,0], O2[-0.524,1.124,0]
- COM [0.249,0.391,0]; R_g=0.924Å
- σ_bead (CG) = 2*0.924 + 0.45*3.13 + 0.15 + 0.055 ≈3.5Å
- Geometry Effect: Bent (115°) → medium R_g; linear → +0.3Å σ.

### S=O (Linear, double bond 1.43Å)
- All-Atom Positions: S[0,0,0], O[1.43,0,0]
- COM [0.476,0,0]; R_g=0.674Å
- σ_bead (CG) = 2*0.674 + 0.5*3.31 + 0.1 ≈3.0Å
- If single bond (1.63Å): R_g=0.72Å → σ≈3.3Å (+0.3Å).

### SO₂ (Bent, two double bonds, angle 120°)
- All-Atom Positions: S[0,0,0], O1[1.43,0,0], O2[-0.715,1.239,0]
- COM [0.179,0.309,0]; R_g=0.945Å
- σ_bead (CG) = 2*0.945 + 0.45*3.31 + 0.1 ≈3.2Å
- Geometry: Bent → compact; tetrahedral → +1.5Å σ.

**Geometry Effects Summary**:
| Geometry | R_g Factor | σ Example |
|----------|------------|-----------|
| Linear | High | 3.0-3.8Å |
| Bent | Medium | 3.2-3.5Å |
| Tetrahedral | Low | 4.5-4.7Å |

## 4. Decision Logic (Python Pseudocode)
```python
def classify_geometry(n_atoms, angles):
    if n_atoms == 2: return "LINEAR"
    elif n_atoms == 3:
        avg_angle = sum(angles) / len(angles)
        if avg_angle > 170: return "LINEAR"
        elif 100 < avg_angle < 130: return "BENT"
    elif n_atoms == 4:
        if is_planar(angles): return "TRIGONAL_PLANAR"
        else: return "TETRAHEDRAL"
    return "COMPLEX"

def get_flexibility(bond_orders):
    avg_order = sum(bond_orders) / len(bond_orders)
    if avg_order >= 2.5: return 0.05
    elif avg_order >= 1.8: return 0.10
    elif avg_order >= 1.3: return 0.15
    else: return 0.25

def get_vdw_weight(geometry, n_atoms):
    weights = {"LINEAR": 0.50, "BENT": 0.45, "TETRAHEDRAL": 0.40, "TRIGONAL_PLANAR": 0.48, "COMPLEX": 0.35}
    return weights.get(geometry, 0.40)

def estimate_sigma(atoms, bonds, angles):
    positions = build_3d(atoms, bonds, angles)  # Use bond lengths/angles to place atoms
    masses = [a['mass'] for a in atoms]
    com = sum(m * p for m, p in zip(masses, positions)) / sum(masses)
    rg_sq = sum(m * np.linalg.norm(p - com)**2 for m, p in zip(masses, positions)) / sum(masses)
    rg = np.sqrt(rg_sq)
    sigma_avg = sum(a['sigma'] for a in atoms) / len(atoms)
    bond_orders = [b['order'] for b in bonds]
    geometry = classify_geometry(len(atoms), [a['value'] for a in angles])
    w_vdw = get_vdw_weight(geometry, len(atoms))
    f_flex = get_flexibility(bond_orders)
    f_elec = 0.02 * (sum(bond_orders)/len(bonds) - 1) * (2 * rg)
    sigma_bead = 2 * rg + w_vdw * sigma_avg + f_flex + f_elec
    return round(sigma_bead, 2)
```

## 5. Calculation of Rmin from estimated effective sigma of a coarse-grained (CG) bead
**Core Idea**: Rmin is the distance at which the LJ potential reaches its minimum. It is related to the effective σ_bead by the equation: `Rmin = σ_bead * 2^(1/6)`.
**Calculation**:
- Rmin = σ_bead * 2^(1/6), where 2^(1/6) ≈ 1.1225
- Example: For a bead with σ_bead = 3.5Å, Rmin = 3.5 * 1.1225 ≈ 3.95Å

## 6. Bond length estimation from sigma values of two beads (not to be confused with Rmin of a single bead)
**Core Idea**: The bond length between two CG beads is related to their effective σ values by the equation: `bond_length = 0.5 * (σ1 + σ2)`.
**Calculation**:
- Bond length = 0.5 * (σ1 + σ2) 
- Example: For two beads with σ1 = 3.0Å and σ2 = 3.5Å, bond length = 0.5 * (3.0 + 3.5) = 3.25Å

## 7. Key Takeaways & Equations
- Bond length/order dominate σ: Longer/single → +0.2-0.4Å σ → ↓ density.
- Geometry: Linear → largest σ; compact → smallest.
- Never direct all-atom transfer: CG softer (k 1-10 vs. 100-500), larger ε (0.5-2 vs. 0.1-0.2).
- Equations:
  - LJ: `U(r) = ε[(Rmin/r)¹² - 2(Rmin/r)⁶]`
  - ΔH_vap ≈ N * ε * z / 2 (z=coordination)
  - RDF g(r): Local density; peak pos ≈ Rmin, height ∝ ε.
  - R_g² = Σ m_i |r_i - COM|² / Σ m_i
  - σ_bead = 2 * R_g + w_vdw * σ_avg + f_flex + f_elec
  - bond_length = 0.5 * (σ1 + σ2)
  - Rmin = σ_bead * 2^(1/6)
